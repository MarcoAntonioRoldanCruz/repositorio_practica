Guía para Manejar Repositorios Git "Diana Ayala"

Para clonar un repositorio, abre tu terminal y ejecuta el siguiente comando:

git clone https://github.com/MarcoAntonioRoldanCruz/repositorio_practica

Reemplaza usuario y repositorio con los datos correspondientes.

Una vez clonado el repositorio, navega al directorio del mismo:

cd repositorio

Crea una nueva rama para tus cambios:

git checkout -b mi-nueva-rama

Edita los archivos necesarios utilizando tu editor de texto preferido.

Añade los archivos modificados al staging area:

git add .

Confirma los cambios con un mensaje descriptivo:

git commit -m "Descripción de los cambios"

Envía los cambios a tu repositorio remoto:

git push origin mi-nueva-rama
